# ERA5 Temperature Prediction using LSTM

## Overview

This project predicts daily temperature for major Indian cities using Long Short-Term Memory (LSTM) neural networks.

The temperature data is obtained from the ERA5 reanalysis dataset.

## Objectives

- Extract temperature data for Indian cities
- Preprocess and normalize data
- Create sliding-window sequences
- Train LSTM models
- Evaluate forecasting performance

## Cities

- Delhi
- Mumbai
- Bengaluru
- Hyderabad
- Chennai
- Kolkata
- Jaipur
- Ahmedabad
- Surat 
- Pune
